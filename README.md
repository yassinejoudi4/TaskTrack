
# TaskTrack – Gestion de tâches

Petit projet web réalisé en HTML, CSS et JavaScript pour gérer des tâches personnelles ou professionnelles.

## Fonctionnalités

- Ajouter de nouvelles tâches avec un titre et une description.  
- Modifier ou supprimer des tâches existantes.  
- Marquer les tâches comme “faites” ou “à faire” en un clic.  
- Interface simple et intuitive, mise à jour dynamique sans recharger la page.  
- **Attention : les tâches ne sont pas sauvegardées. Si vous rafraîchissez la page, elles seront perdues.**

## Technologies

- HTML, CSS, JavaScript  
- Pas de stockage permanent pour le moment (données en mémoire seulement)  

## Objectif

- Projet simple mais utile pour gérer ses tâches, facilement améliorable.  
- Pratiquer le développement front-end basique et la manipulation du DOM.  

## Idées et améliorations futures

**Les améliorations futures sont en cours de travail :**  

- Ajouter un stockage permanent des tâches avec `localStorage` ou une base de données.  
- Ajouter la possibilité de trier ou filtrer les tâches par priorité ou catégorie.  
- Ajouter un système de connexion pour plusieurs utilisateurs.  
- Ajouter des notifications ou rappels pour les tâches importantes.  
- Améliorer l’interface avec des animations ou un design responsive.  

## Objectif pédagogique

- Comprendre la manipulation du DOM avec JavaScript.  
- Pratiquer la structuration d’un projet web simple.  
- Préparer le projet pour des évolutions futures et des fonctionnalités avancées.
